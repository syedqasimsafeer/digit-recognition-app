
# Digit Recognition App Using NumPy and PyTorch

## Project Overview

This project is a complete educational deep learning project for handwritten digit recognition.

It includes:

- MNIST dataset loading
- Image visualization
- Neural network from scratch using NumPy
- Backpropagation explanation
- MLP model using PyTorch
- CNN model using PyTorch
- Model evaluation
- Confusion matrix
- Wrong prediction analysis
- Streamlit app deployment

## Dataset

The project uses the MNIST handwritten digit dataset.

MNIST contains grayscale digit images from 0 to 9.
